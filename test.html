Testing Subscript and Superscript

This is an R Markdown document.

Pandoc includes numerous extensions to markdown, and one of them is subscript and superscript.

Here's the example from the Pandoc help page (http://johnmacfarlane.net/pandoc/README.html#superscripts-and-subscripts): H~2~O is a liquid. 210^ is 1024.

For fun, here's an R code block with some code from @Spacedman:

list.depth <- function(this, thisdepth = 0) {
# http://stackoverflow.com/a/13433689/1270695
  if(!is.list(this)) {
    return(thisdepth)
  } else {
    return(max(unlist(lapply(this, list.depth, thisdepth = thisdepth+1))))    
  }
}
